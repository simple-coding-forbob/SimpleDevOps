# 02_계정만들기.md (관리자가 할 일)
# 관리자 계정: root
## 계정 / 그룹: 
- 계정: 사용자
- 그룹: 계정을 목적 또는 권한별로 분류해 놓은 것

# 예제) 개발자(DB 관리) 계정 만들기
# 1) 그룹 생성: dba
#  사용법: addgroup 그룹명
[root@localhost ~]# addgroup dba

# 2) 계정 생성: oracle
##### 사용법 : adduser 계정명 -s 셀 -h 작업폴더 -G 그룹명
#####      -s : 셀환경 선택(바탕화면테마와 비슷) , /bin/bash(배쉬 셀)
#####      -h : 홈폴더(==디렉토리) , 사용자별 작업공간폴더를 의미
#####      -G : 그룹 지정
##### 예) 셀 종류: 바탕화면 테마와 비슷, bash(추천), csh 등
[root@localhost ~]# adduser oracle -s /bin/bash -h /home/oracle -G dba

# 3) 암호 설정: 
##### 암호 설정 시 입력 글자가 안보임
##### 패스워드 : !Ds1234567890
##### 사용법 : passwd 계정명
##### 참고) 화면 지우기 명령어: ctrl + l
[root@localhost ~]# passwd oracle
Changing password for user oracle.
New password:
BAD PASSWORD: The password fails the dictionary check - it is too simplistic/systematic(무시)
Retype new password:
passwd: all authentication tokens updated successfully.

# 4) 계정 바꾸기: root(관리자) -> oracle
##### 사용법 : su - 계정명
##### (참고) # -> 관리자 계정, $ -> 일반 계정
[root@localhost ~]# su - oracle
[oracle@localhost ~]$

# 5) 계정 삭제: root 계정 접속 후 실행 
##### 사용법 : deluser 계정명
[root@localhost ~]# deluser oracle
