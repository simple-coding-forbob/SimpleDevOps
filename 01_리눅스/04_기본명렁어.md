# 04_기본명령어.md

# 1) 현재 접속된 유저 정보 보기
##### 사용법 : id
[oracle@localhost ~]$ id
uid=1000(oracle) gid=1000(dba) groups=1000(dba)

# 2) 현재 날짜
##### 사용법 : date
[oracle@localhost ~]$ date
Mon Sep  8 03:05:19 AM EDT 2025

# 3) 화면에 글자 hello 표시하기
##### 사용법 : echo 문자열
[oracle@localhost ~]$ echo hello
hello
[oracle@localhost ~]$ echo hello world
hello world

##### (참고) 환경변수: echo $SHELL
##### (참고) 환경변수: echo $PATH
##### 셀 변수: bash
##### 경로(PATH) 변수: $PATH
##### (참고) $PATH 에 프로그램 경로를 넣어두면 어디에서든 실행되어 편리함
[oracle@localhost ~]$ echo $SHELL
/bin/bash
[oracle@localhost ~]$ echo $PATH
/home/oracle/.local/bin:/home/oracle/bin:/usr/local/bin:/usr/bin:/usr/local/sbin:/usr/sbin

# 5) 명령어 자동완성 기능: tab
[oracle@localhost ~]$ da(tab 키 입력)

# 6) 이전 명령어 찾기 기능: 화살표 키 (위, 아래)
# 화살표 키 위: 이전 명령어
# 화살표 키 아래: 최근 명령어
[oracle@localhost ~]$ echo $SHELL (화살표 키 위)
