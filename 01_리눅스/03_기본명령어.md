# 03_기본명령어.md (개발자가 할 일)

# 1) 현재 경로 보기
##### 현재 경로(path) 보기 
##### 사용법 : pwd
##### /(슬래쉬): 폴더 구분자로 사용
##### 폴더(윈도우)==디렉토리(리눅스)
[oracle@localhost ~]$ pwd
/home/oracle

# 2) 폴더나 파일 목록 보기
##### 사용법 : ls -al [== ls -a -l]
##### -a : 숨긴파일까지 모두 보기 (.파일명 : 숨긴파일)
##### -l : 자세히 보기 (권한, 사이즈 등)
##### (참고) .파일명 : 숨김파일
[oracle@localhost ~]$ ls
[oracle@localhost ~]$ ls -al
