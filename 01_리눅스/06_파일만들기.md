# 06_파일만들기.md
# 리눅스 : 에디터(메모장 같은것) 사용(nano)
# 에디터 종류 예) vi(vim), nano 등
### 1) 파일 만들기
### 사용법: nano 파일명
### 2) 파일 저장: ctrl + o
### 3) 나가기   : ctrl + x
### 예제1) hello.txt 만들기(내용: hello world)
[root@localhost ~]# ls -l
total 12
-rw-r--r-- 1 root root 114 Dec 26  2020 bench.py
-rw-r--r-- 1 root root 185 Sep  9  2018 hello.c
-rw-r--r-- 1 root root  13 Sep  9 10:20 hello.txt

### 2) 파일 복사하기
### 예제2) hello.txt 파일을 복사해서 hello2.txt 를 만드세요
##### 사용법 : cp 원본파일명 복사파일명
##### cp(copy)
[root@localhost ~]# ls
bench.py  hello.c  hello.txt
[root@localhost ~]# cp hello.txt hello2.txt
[root@localhost ~]# ls
bench.py  hello2.txt  hello.c  hello.txt

### 퀴즈2) hello2.txt 파일을 복사해서 hello3.txt 를 만드세요
[root@localhost ~]# cp hello2.txt hello3.txt
[root@localhost ~]# ls
bench.py  hello2.txt  hello3.txt  hello.c  hello.txt

### 3) 파일 이동하기
### 예제 3) hello.txt 를 newfolder 로 이동시키세요
###  단, newfolder 없으면 만드세요
### 새폴더 만들기: mkdir 새폴더명
##### 사용법 : mv 파일명 폴더명
##### mv(move)
[root@localhost ~]# ls
bench.py  hello2.txt  hello3.txt  hello.c  hello.txt
[root@localhost ~]# mkdir newfolder
[root@localhost ~]# ls
bench.py  hello2.txt  hello3.txt  hello.c  hello.txt  newfolder
[root@localhost ~]# mv hello.txt newfolder/
[root@localhost ~]# ls
bench.py  hello2.txt  hello3.txt  hello.c  newfolder
[root@localhost ~]# cd newfolder
[root@localhost newfolder]# ls
hello.txt

### 3) 파일명 바꾸기
### 예제 3) hello2.txt 를 newhello.txt  로 이름 바꾸시오
##### 사용법 : mv 파일명 변경파일명
[root@localhost ~]# ls
bench.py  hello2.txt  hello3.txt  hello.c  newfolder
[root@localhost ~]# mv hello2.txt newhello.txt
[root@localhost ~]# ls
bench.py  hello3.txt  hello.c  newfolder  newhello.txt

### 4) 파일 삭제
### 예제 4) newhello.txt 삭제하세요
##### 사용법) rm 파일명 파일명2 ....
[root@localhost ~]# ls
bench.py  hello3.txt  hello.c  newfolder  newhello.txt
[root@localhost ~]# rm newhello.txt
[root@localhost ~]# ls
bench.py  hello3.txt  hello.c  newfolder

### 예제5) hello3.txt 간단 읽기 하세요
### 사용법: cat 파일명
