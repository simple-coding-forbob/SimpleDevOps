# 폴더(디렉토리)
# 폴더 구조 설명: 계층형 구조
    (1) 경로(Path) : /(슬래쉬) 사용해서 폴더 경로 표시함
        예) /home/oracle => home 밑에 oracle 폴더
            (vs 윈도우 명령프롬프트 : \(역슬래쉬))
    (2) 경로의 최상위 경로 : / => 루트(root)경로
    (3) 리눅스를 설치하면 시스템 폴더(관리 폴더)
        /bin : 주요 명령어 들이 있는 폴더
        /etc : 계정, 그룹, 패스워드 등의 환경설정 파일 있음
        /home: 계정별로 할당되는 작업폴더
        /sbin: 슈퍼 관리자가 사용하는 명령어들 있음
            (컴퓨터 끄기, 켜기 등)
        /tmp : 임시파일이 잠시 머무르는 폴더
        /usr : 설치된 프로그램이 있는 폴더
        /var : 로그(접속, 시스템감시 등)가 있는 폴더

# 예제 1) 경로 이동: a폴더 -> b폴더 이동
##### (1) 예제) /usr 경로 이동
##### 사용법 : cd 폴더명
##### cd == change directory(폴더)
[root@localhost ~]# cd /usr
[root@localhost usr]# pwd
/usr

##### (2) 예제) /usr -> / 이동
[root@localhost usr]# cd /
[root@localhost /]# pwd
/

##### (3) 예제) 상위폴더로 이동하기
##### 예제 3) /usr/lib -> /usr 으로 이동하기(상위경로)
##### .. : 상위폴더(디렉토리), . : 현재폴더(디렉토리)
##### 사용법 : cd ..
[root@localhost lib]# pwd
/usr/lib
[root@localhost lib]# cd ..
[root@localhost usr]# pwd
/usr

##### 예제 4) /usr -> /usr/lib 으로 다시 이동하기(이전 폴더로 이동하기)
##### 사용법 : cd -
[root@localhost usr]# pwd
/usr
[root@localhost usr]# cd -
/usr/lib
[root@localhost lib]# cd -
/usr

##### 예제 5) 새폴더(디렉토리) report 만들기
##### 사용법 : mkdir 폴더명
##### mkdir(make directory)
##### root(관리자계정) : /root(root 작업폴더)
[root@localhost ~]# cd /root
[root@localhost ~]# mkdir report
[root@localhost ~]# ls
bench.py  hello.c  report

##### 예제 6) report 빈폴더 삭제하기
##### 사용법: rmdir 폴더명
##### 단, 빈폴더만 삭제가능
##### rmdir(remove directory)
[root@localhost ~]# rmdir report
[root@localhost ~]# ls
bench.py  hello.c

##### 예제 7) 폴더 목록 정보 설명: ls -al
##### drwxr-xr-x 부분 :
##### d : 디렉토리(폴더), - : 파일
##### l : 심볼릭링크(윈도우의 바탕화면 바로가기 아이콘)
##### r : read(읽기권한)
##### w : write(쓰기권한)
##### x : excute(실행권한)
##### rwx              r-x           r-x
##### 예) -rwxrwxrwx
##### (만든사람(user)) (그룹(group)) (다른사람(other))
