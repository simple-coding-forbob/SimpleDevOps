# 1) Docker Desktop 설치 (필수)
https://www.docker.com/products/docker-desktop/

# 2) 확인
docker --version

# 3) kubectl 설치
PowerShell에서 실행
curl -LO "https://dl.k8s.io/release/v1.29.0/bin/windows/amd64/kubectl.exe"

# 4) 환경변수 추가 (중요)
kubectl.exe 있는 폴더를 PATH에 추가

# 5) 설치 확인
kubectl version --client

# 6) Minikube 설치: 
https://minikube.sigs.k8s.io/docs/start/ 

# 7) Minikube 실행: 
docker 실행: docker desktop 실행
docker 실행 확인: docker ps
minikube 실행 :minikube start —driver=docker
