# (참고) 01_요약.md
# 목적: 도커 때문에 나오게 된 프로그램
# 도커로 컨테이너를 여러개 만들다보면 개수가 엄청 늘어납니다.
# 그래서, 컨테이너를 목적별로 묶어서 관리해주는 프로그램이 필요해 졌습니다.
# 그래서 요즘 실무에서 쿠버네티스를 도입하게 됩니다.(왕창 실행, 왕창 끄기)
# 추가적인 옵션 기능도 있습니다.
# 용어 정리: 1) 파드(pod)             : 컨테이너들이 포함되어 있는 곳, 독립적으로 동작합니다.
#           2) 서비스                : 웹브라우저 <-> 파드(pod) 와의 쉬운 통신을 위해 사용합니다.
#           3) 배포(deployment) 문서 : 파드/서비스를 만들고 자동실행해주는 문서
